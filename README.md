# 기분샵

## 1. 용도
- 기분이 꿀꿀할 때 사용
- 사기칠 때 사용(보이스피싱)

## 2. 사용방법
검색-장바구니에 담기-구매

## 3. 기능
- 홈화면 오늘의 특가(할인 특가의 물건을 추천)
- 배송조회(실제로 안옴)
- 뭐살까 추천 봇(GEMINI)
- 찜 기능(위시리스트)
- 가챠/룰렛뽑기

## 문제가 생기면
- **"네이버 API 호출에 실패했어요" 에러**: 서버 폭발한 거임. 
- **"서버에 연결할 수 없어요"**: 좀 기다리면 됨. (개발자 지갑 사정 문제)
- **"아무것도 상호작용이 안돼요" 문제**: 당신 문제임ㅇㅇ.
