# 기분샵

## 1. 용도
- 기분이 꿀꿀할 때 사용
- 사기칠 때 사용(보이스피싱)

## 2. 사용방법
압축 푼 폴더로 터미널에서 이동:
```
cd naver-mood-shop
```

## 3. 필요한 패키지 설치
```
npm install
```
(express, dotenv 두 개만 설치돼요. 몇 초면 끝나요.)

## 4. .env 파일 만들기
`.env.example` 파일을 복사해서 `.env`라는 이름으로 저장하고,
그 안에 있는 값을 진짜 Client ID / Secret으로 바꿔주세요.

터미널로 하려면:
```
cp .env.example .env
```
그 다음 `.env` 파일을 텍스트 에디터로 열어서 수정.

`.env` 내용 예시:
```
NAVER_CLIENT_ID=abcd1234실제값
NAVER_CLIENT_SECRET=efgh5678실제값
PORT=3000
```

## 5. 서버 실행
```
npm start
```
터미널에 `✅ 서버 실행 중! http://localhost:3000` 이렇게 뜨면 성공.

## 6. 브라우저에서 열기
`http://localhost:3000` 접속 → 검색창에 아무거나 검색 (예: "무선이어폰")

## 문제가 생기면
- **"네이버 API 호출에 실패했어요" 에러**: `.env`에 키가 정확히 들어갔는지, 네이버 개발자센터에서 이 앱에 "검색" API 권한이 켜져있는지 확인
- **"서버에 연결할 수 없어요"**: 터미널에서 서버가 계속 실행 중인지 확인 (껐으면 `npm start` 다시)
- **포트 3000이 이미 사용 중이라는 에러**: `.env`에서 `PORT=3001`처럼 바꿔보기

## 이 코드가 하는 일 / 안 하는 일
- ✅ 네이버 쇼핑 API로 진짜 상품 검색 결과를 가져옴
- ✅ 장바구니 담기, 삭제
- ✅ "구매하기" 누르면 축하 애니메이션 + 가짜 영수증
- ❌ 실제 결제 절대 안 함 (결제 API 자체를 안 씀)
- ❌ 실제 배송 당연히 안 됨
- ❌ 새로고침하면 장바구니 사라짐 (저장 기능 없음 — 원하면 추가 가능!)
